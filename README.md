# Tic Tac Toe Game

## Overview

Tic Tac Toe is a classic two-player game where players alternate turns marking Xs and Os on a 3x3 grid. This project brings the game to life with a modern and dynamic user interface built using ReactJS. 

### Features

- **Two-player Gameplay**: Players can enter their names and choose their symbols (X or O).
- **Dynamic UI**: Beautiful animations for player symbol selection and game actions.
- **Winning Conditions**: Highlights winning squares for clear visual feedback.
- **Background Image**: Customize your gameplay experience with a stunning background image.
- **Responsive Design**: Playable on various devices with a smooth experience.

## Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Saubhik998/tic-tac-toe.git
   ```

2. **Navigate to the Project Directory**
   ```bash
   cd tic-tac-toe
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Start the Development Server**
   ```bash
   npm start
   ```

   Your application will open in the browser at `http://localhost:3000`.

## Usage

- Upon starting the game, enter the names of both players.
- The first player chooses their symbol (X or O), with the second player getting the alternate symbol.
- Players take turns clicking on the squares to place their symbols.
- The game highlights the winning squares and shows the winner at the end.
- Restart the game anytime to play again!
